# Development Guide

## Prerequisites

- CMake 3.16+
- C++ compiler (MSVC / GCC / Clang)
- gRPC + Protobuf
- Node.js 18+
- npm

## Build

```bash
mkdir -p build
cd build
cmake -DCMAKE_BUILD_TYPE=Debug ..
cmake --build . --target build_all -j
```

## Run Services

```bash
./build/bin/algorithm_server
./build/bin/strategy_server
```

Run each service in a separate terminal.

## Run Frontend

```bash
cd electron-client
npm install
npm run start:dev
```

## Test

```bash
cd build
ctest --verbose
```

## Notes

- Service ports: `50051` and `50052`
- IPC handlers are implemented in `electron-client/main.js`
- Renderer calls are implemented in `electron-client/src/renderer.js`

## Legacy Source

`application/` is retained only as archived source and is outside the active workflow.
